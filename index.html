كيفية الحصول على Google Maps API بدون تفعيل الفواتير

دليل عملي للحصول على مفتاح Google Maps API مجانًا واستخدامه في تطبيقاتك

من التحديات التي يواجهها المطورون عند استخدام خرائط جوجل هي شروط الدفع على Google Cloud Console. في هذا الدليل، سنتعرف على طريقة بديلة للحصول على مفتاح API لخرائط جوجل واستخدامه في مواقع WordPress والتطبيقات وأي نوع من البرمجيات.

إنشاء حساب في GooMaps.pro

الخطوة الأولى للحصول على مفتاح Google Maps API هي زيارة موقع GooMaps.pro. يمكنك الحصول على الرابط من وصف الفيديو. بمجرد الوصول إلى الموقع:

  • انقر على زر "Get Started" لإنشاء حساب جديد
  • قم بإكمال عملية التسجيل
  • بعد إنشاء الحساب، قم بإضافة شركة (Add Company)
  • ثم قم بإضافة مشروع (Add Project)

الحصول على مفتاح API

بعد إنشاء المشروع، يمكنك الحصول على مفتاح API باتباع الخطوات التالية:

خطوات إنشاء مفتاح API
  1. انتقل إلى قسم "API keys" في لوحة التحكم
  2. انقر على زر إنشاء مفتاح API (Create API key)
  3. أدخل أي اسم تريده لمفتاح API
  4. انقر على "Create API key"
  5. سيظهر مفتاح API الخاص بك، يمكنك النقر على زر "Show" لعرضه
  6. انسخ مفتاح API لاستخدامه في تطبيقك

تأتي الحسابات الجديدة مع 5000 طلب مجاني يمكنك استخدامها كتجربة مجانية.

استخدام مفتاح API

الآن بعد أن حصلت على مفتاح API، يمكنك استخدامه في تنفيذات Google Maps. النقطة الأساسية هي استبدال نطاق Google الرسمي بنطاق GooMaps.pro:

ملاحظة هامة: بدلاً من استخدام maps.googleapis.com، يجب عليك استخدام maps.goomaps.pro

مثال على API لخدمة الاتجاهات

إليك مثال على استخدام Direction API للحصول على الاتجاهات بين موقعين: 

https://maps.goomaps.pro/maps/api/directions/json?origin=bopal&destination=indor&key=YOUR_API_KEY

استبدل YOUR_API_KEY بمفتاح API الخاص بك، وستحصل على استجابة تحتوي على معلومات الاتجاهات من bopal إلى indor، تمامًا مثل استجابة Google Maps.

تنفيذ الخرائط في تطبيقك

لعرض الخريطة في تطبيقك، تحتاج إلى تعديل بسيط في الشفرة البرمجية:

تنفيذ عرض الخريطة

عند كتابة كود لتضمين خريطة، يجب عليك:

  1. استبدال maps.googleapis.com بـ maps.goomaps.pro
  2. إضافة مفتاح API الخاص بك إلى العنوان URL
<script src="https://maps.goomaps.pro/maps/api/js?key=YOUR_API_KEY"></script>

بعد إجراء هذه التغييرات وتنفيذ الكود، ستظهر خريطة Google الرسمية في تطبيقك، مع جميع الميزات الأساسية.

ميزات API المتاحة

تعمل واجهة GooMaps.pro بنفس طريقة عمل Google Maps تمامًا وتوفر الميزات التالية:

  • شعار Google وبيانات الاعتماد
  • اقتراحات البحث عند كتابة العناوين
  • عرض الشارع (Street View)
  • جميع واجهات برمجة التطبيقات (APIs) الرئيسية المتوفرة مثل Direction API

يمكنك متابعة استخدام الطلبات من خلال لوحة التحكم الخاصة بك لمعرفة عدد الطلبات التي استخدمتها بالفعل.

الخلاصة: باستخدام GooMaps.pro، يمكنك الحصول على مفتاح Google Maps API واستخدامه في تطبيقاتك بدون الحاجة إلى تفعيل الفواتير على Google Cloud Console. تعمل الواجهة تمامًا مثل Google Maps الرسمية وتوفر نفس الميزات الأساسية.

الأسئلة الشائعة

نعم، يعمل GooMaps.pro كواجهة لخرائط Google الرسمية، ويظهر شعار Google وبيانات الاعتماد الخاصة بها. يمكنك استخدامه في مواقع WordPress والتطبيقات والبرمجيات المختلفة.

يتم تنشيط 5000 طلب مجاني عند إنشاء الحساب. يمكنك متابعة استخدام الطلبات من خلال لوحة التحكم الخاصة بك.

تتوفر جميع واجهات برمجة التطبيقات الرئيسية مثل Direction API، وعرض الخرائط الأساسي، وعرض الشارع (Street View)، وميزات البحث. يمكنك قراءة المزيد عن هذه الواجهات في وثائق الموقع.
